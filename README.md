# Rover for Reddit

This project aims to recreate the [Apollo](https://apolloapp.io/) for web and as a PWA.

Rover is built in [Svelte](https://svelte.dev/)

## Features
- Fully fledged comment tree view
- Search page
- Home, Subreddit, and User views
